# Tigerverse-Hackathon-2026---Spectacles---Scene-Blocker
This is a project for the 2026 Tigerverse Hackathon. It is a lens that goes through a scene line by line, ans assigns locations for blocking. It moves on from each line via voice recognition.

You can edit each slide directly to get the line you want to say, and then you go into the voicecontroller script to edit the phrases the voice recognition will detect for.

A majority of the scripts were written alongside Vibe-coding AI, spesifically Cursor. I would use it to write these scripts and tweak internal values, while I created all of the visual textures, lines, and blocking markers.

I used a couple of presets taken from the official Spectacles github link, as well as context I fed into Cursor. A big majority came from the Path Pioneer and Public Speaking examples spesifically.
